# **App Name**: Supermoda Coupons

## Core Features:

- Employee Registration: Employees can register with their details.
- Sales Entry: Employees can enter their sales data, including purchase number, value, and date.
- Coupon Generation: The app automatically calculates and generates coupons based on the sales amount (e.g., one coupon for every R$ 250.00).
- Admin Dashboard: Admin can access and manage all generated coupons.
- Raffle Management: The admin can perform raffle at the end of the campaign using an AI tool for randomness.
- Data Storage: Sales data and generated coupons are stored securely in the database.
- Countdown timer: Displays the end date of the Supermoda sales promotion to the user.

## Style Guidelines:

- Primary color: Vivid pink (#E91E63) to reflect the energy and excitement of Supermoda's sales promotions.
- Background color: Light pink (#F8BBD0), slightly desaturated, creates a soft and inviting atmosphere.
- Accent color: Deep rose (#C2185B) will be used for primary CTAs, such as the 'Generate Coupon' button.
- Body and headline font: 'PT Sans', a friendly sans-serif, ensuring clarity and readability.
- Use clear, modern icons to represent different actions (e.g., submitting sales, viewing coupons). Consider line-based icons with rounded corners.
- Clean, spacious layout. Forms and key information should be easy to find, well-organized and self-explanatory.
- Subtle animations to give positive feedback, such as when the user sucessfully adds sales data.
